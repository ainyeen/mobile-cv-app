Students Attendance (Java)



This is a Java-based console application that calculates students' attendance percentages based on the number of school days.

It allows users to input each student's name and the number of days they were present.

The program then calculates their attendance rate and automatically flags students with less than 85% attendance with a "Warning Letter".

It also keeps track of the total number of students and how many received warnings.
